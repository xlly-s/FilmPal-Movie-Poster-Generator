# FilmPal: The Movie Poster Generator
Uses python and PIL to create movie posters in command line.


## I'd like to say a massive thank you to Anand Murphy (https://github.com/anand-murthy-r) for helping me get it get it setup and talking me through the process. Plus the original code and ideas which I then updated and changed.


#Instructions For Usage

1. Download the files and open them in your wanted envirment (Tested on windows and linux)
2. You will need a TMDB API key, you can read how to get one here: https://developer.themoviedb.org/docs/getting-started
3. Once signed up, goto https://www.themoviedb.org/settings/api and copy your api key, NOT THE READ ACCESS TOKEN!
4. Paste it into the script at TMDB_API_KEY = "api_key" and save
5. Run 'FilmPal.py' and enter a movie
6. Select the style (Enter auto selects designer)
7. Select the movie you want
8. Once done is will open the poster which you can then promptly save anywhere!
9. Have fun!

[For some reason is hates obsure movie names like 'WALL·E' with the dot, I had to manually copy and paste it in from TMDB website. I'll Look to change this later.]
